# university-project-work-vault
How to start simulation of this project,
in the terminal write:

python setup_azienda.py

python database.py

python main_simulation.py

python report_analytics.py

Note: The file anagrafica_prodotti.xlsx is used to import product data and simulate a real-world context. This allows the final report to be based on real, processed data, not random data generated by the program itself.
